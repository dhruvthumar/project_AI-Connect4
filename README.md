# Connect 4 Pattern Recognition

Academic Project for CS4795 : Introduction to Artificial Intelligence  
Pattern recognition on Connect 4 board states using the UCI dataset.

## Setup
pip install pandas numpy scikit-learn matplotlib seaborn ucimlrepo

## Run
1. Convert: 
            jupyter nbconvert --to python main.ipynb (This creates a file named main.py)
2. Run: 
            python main.py

## Dataset
UCI Connect-4 Dataset (67,557 board positions)
https://archive.ics.uci.edu/dataset/26/connect+4
